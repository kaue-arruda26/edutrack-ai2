# OpenSpec Agent Rules

## Core Rules
1. Every change must start with /opsx:new.
2. Follow the sequence: Proposal -> Specs -> Tasks -> Apply.
3. Keep it simple: Only do what the user explicitly requested.

## Mandatory Files
- openspec/changes/<change-name>/proposal.md
- openspec/changes/<change-name>/specs/spec.md
- openspec/changes/<change-name>/tasks.md

## XanoScript Rules
- All table definitions must be in the `tables/` folder.
- Follow XanoScript syntax for .xs files.
